=== Console 에이전트 4.5.0

4.5.0 릴리스는 표준 모드와 제한된 모드를 모두 지원합니다.

이번 Console 에이전트 릴리스에는 보안 개선 사항 및 버그 수정 사항과 더불어 다음과 같은 새로운 기능이 포함되어 있습니다.

.필수 포트의 구성 검사 지원
구성 검사를 사용하여 Console 에이전트에 필요한 포트가 열려 있는지 확인할 수 있습니다. Console 또는 Agent 유지 관리 콘솔에서 구성 검사를 수행할 수 있습니다.link:https://docs.netapp.com/us-en/console-setup-admin/task-troubleshoot-agent.html["Console 에이전트에서 구성 검사를 실행하는 방법에 대해 자세히 알아보세요."]



=== 콘솔 관리

파트너 사용자에게 Keystone 역할을 할당하여 해당 사용자가 파트너 조직의 Keystone 리소스를 관리할 수 있도록 할 수 있습니다.link:https://docs.netapp.com/us-en/console-setup-admin/task-partnerships-assign-users.html["파트너 사용자에게 Keystone 역할을 할당하는 방법에 대해 자세히 알아보세요."]
