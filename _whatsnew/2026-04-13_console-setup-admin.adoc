=== Console 에이전트 4.6.0

4.6.0 릴리스는 표준 모드와 제한된 모드를 모두 지원합니다.

이번 Console 에이전트 릴리스에는 보안 개선 사항 및 버그 수정 사항과 더불어 다음과 같은 새로운 기능이 포함되어 있습니다.

.필요한 디스크 공간의 구성 확인 지원
구성 검사를 사용하여 Console 에이전트에 필요한 디스크 공간을 사용할 수 있는지 확인할 수 있습니다. Console 또는 Agent 유지 관리 콘솔에서 구성 검사를 수행할 수 있습니다.link:https://docs.netapp.com/us-en/console-setup-admin/task-troubleshoot-agent.html["Console 에이전트에서 구성 검사를 실행하는 방법에 대해 자세히 알아보세요"^]

.ISO를 사용하여 vCenter에 배포된 Console 에이전트 업그레이드
OVA를 사용하여 vCenter에 Console 에이전트를 배포한 경우, 이제 ISO를 VM에 마운트하고 VM 유지 관리 콘솔에서 업그레이드를 실행하여 새 버전으로 업그레이드할 수 있습니다. link:https://docs.netapp.com/us-en/console-setup-admin/task-upgrade-agent.html["vCenter에 배포된 콘솔 에이전트를 업그레이드하는 방법을 알아보세요."^]
